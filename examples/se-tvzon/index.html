XMLTV - Ha programtablåer för din digitaltv från och med idag

Här kan du få tablåer för de flesta kanaler för Sverige, även några som man kan se i Sverige, t ex tyska kanaler. Dessa tablåer är för personligt, icke-kommersiellt bruk. Vår data kommer från kanalernas press-sidor eller deras press (eller liknande).
Men vi vill även påminna om att datan vi får kan ibland inte stämma då även kanaler kan skriva fel ibland.
channels / Kanaler
data status / Tablåstatus
logos / Loggor
importer status / Fel-errors
For the nerds / För nördarna:
schedules sent via mail / Tablåer skickade via mail
schedules downloaded via http / Tablåer nerhämtade via webben
